## users テーブル

| カラム名 | 型 | PRIMARY KEY | UNIQUE KEY | NOT NULL | FOREIGN KEY |
| --- | --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | x | x | x |   |
| name | VARCHAR(191) |   |   | x |   |
| email | VARCHAR(191) |   | x | x |   |
| password | VARCHAR(191) |   |   | x |   |
| created_at | DATETIME |   |   |   |   |
| updated_at | DATETIME |   |   |   |   |

- リソースエンティティ
- 会員情報を格納する
- 認証時の識別にはemailを使用する。
- emailはUNIQUEインデックス制約を有効にするために191文字以下に制限している
- passwordはハッシュ化したパスワード

## shift_begins テーブル

| カラム名 | 型 | PRIMARY KEY | UNIQUE KEY | NOT NULL | FOREIGN KEY |
| --- | --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | x | x | x |   |
| user_id | BIGINT UNSIGNED |   |   | x | REFERENCES users(id) |
| begun_at | DATETIME |   |   | x |   |

- イベントエンティティ
- 勤務開始イベントを格納する
- user_idにつき0~1行格納する

## shift_timings テーブル

| カラム名 | 型 | PRIMARY KEY | UNIQUE KEY | NOT NULL | FOREIGN KEY |
| --- | --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | x | x | x |   |
| user_id | BIGINT UNSIGNED |   |   | x | REFERENCES users(id) |
| begun_at | DATETIME |   |   | x |   |
| ended_at | DATETIME |   |   |   |   |

- リソースエンティティ
- 勤務時間の履歴を格納する
- 勤務終了処理を行わずに日を跨いだ場合は、ended_atを`NULL`にしてレコードを格納する
